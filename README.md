# Waxy

[![PyPI](https://img.shields.io/pypi/v/waxy)](https://pypi.org/project/waxy/)

A Python wrapper around the Rust [Taffy](https://docs.rs/taffy/latest/taffy/) UI layout library,
built with [PyO3](https://pyo3.rs/) and [maturin](https://www.maturin.rs/).
